# TapTapConnect

這是一個輕鬆愉快的小作品用於學習藍芽配對、View的建構等等

## 軟體功能

- 可以進行藍芽、WIFI連接
- 上方有按鈕，點擊後可以將目標的的手機震動一下
- 可以藉由此簡單的物件來進行簡單的資訊溝通
